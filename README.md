# ChatApp
to run the code: "nothing to install"
remember to be in the folder you have saved these files!!!
terminal-1: javac ChatServer.java ClientHandler.java ChatClient.java
after runing above one, 
terminal-1: java ChatServer
you will see: "Chat Server started on port 8080"
run how many clients you want.
terminal-2:java ChatClient
after getting GUI, enter your name and start chating.
terminal-3:java ChatClient
after getting GUI, enter your name and start chating.
....
remember not to close any terminal of GUI just minimise them.
you can create as many users you want once all are joined the chats are visible to everyone.
